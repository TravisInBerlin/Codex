nano Cheat Sheet
軽量エディタ nano の“迷わない操作”。よく使う編集・検索・保存・ カット/ペーストをコンパクトに集約。Mac キーボード表記も併記。
Editor Terminal Beginner Friendly
覚えると速い：nano は基本操作がすべて画面下に表示されます。
Mac では Alt が Option に相当します。表記は Alt (Option) で統一。
M- 表記の操作は Esc → キー でも実行できます。
基本操作

ファイル操作

  • nano filename
    ファイルを開く / 新規作成
  • Ctrl + O
    保存（Write Out）
  • Ctrl + X
    終了（保存確認あり）

移動

  • Ctrl + A
    行頭へ移動
  • Ctrl + E
    行末へ移動
  • Ctrl + V
    次ページへ
  • Ctrl + Y
    前ページへ

編集

  • Ctrl + K
    行をカット（切り取り）
  • Ctrl + U
    カットした内容をペースト
  • Ctrl + T
    スペルチェック / コマンド実行
検索・置換

検索

  • Ctrl + W
    検索（次の一致も Enter で続行）
  • Alt (Option) + W
    M-W：次の一致へ（Esc → W でも可）

置換

  • Ctrl + \
    検索して置換
  • Alt (Option) + R
    M-R：置換を繰り返し実行（Esc → R でも可）
便利な小技

ヘルプ/情報

  • Ctrl + G
    ヘルプ表示
  • Ctrl + C
    カーソル位置の行/列を表示
  • Alt (Option) + U
    M-U：Undo（戻る）（Esc → U でも可）
  • Alt (Option) + E
    M-E：Redo（やり直し）（Esc → E でも可）

複数ファイル

  • nano file1 file2
    複数ファイルを開く
  • Alt (Option) + >
    M->：次のファイルへ（Esc → > でも可）
  • Alt (Option) + <
    M-<：前のファイルへ（Esc → < でも可）
Tip: M-（Meta）は Esc で代用可能。Mac では Option を Meta に割り当てると M- 操作が楽になります。

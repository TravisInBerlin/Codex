Vim Cheat Sheet
Vim の頻出モード操作と編集コマンドを実務向けに整理。 ノーマルモード中心に、覚えるだけで速度が段違い。
Editor Modal Productivity
基本は 3 モード：Normal（操作）、Insert（入力）、Visual（選択）。
Esc で Normal に戻れることが最重要の安全装置。
モード & 基本

モード切替

  • i
    Insert モード
  • a
    カーソルの後ろから入力
  • Esc
    Normal に戻る
  • v
    Visual モード（文字単位選択）

保存・終了

  • :w
    保存
  • :q
    終了
  • :wq
    保存して終了
  • :q!
    保存せず終了

移動

  • h j k l
    左/下/上/右
  • 0
    行頭
  • $
    行末
  • gg / G
    先頭 / 最終行へ
編集の基本

削除・変更

  • x
    1 文字削除
  • dd
    行削除
  • cc
    行を変更（削除して Insert）
  • r
    1 文字置換

コピー・貼り付け

  • yy
    行コピー
  • p / P
    下/上に貼り付け
  • u
    Undo
  • Ctrl + r
    Redo

数を付けて高速

  • 3dd
    3 行削除
  • 5j
    5 行下へ
  • 2yw
    2 単語コピー
検索・置換

検索

  • /word
    前方検索
  • n / N
    次/前の一致

置換

  • :%s/old/new/g
    全体置換
  • :s/old/new/g
    行内置換
分割とタブ

分割

  • :split
    横分割
  • :vsplit
    縦分割
  • Ctrl + w
    分割移動のプレフィックス

タブ

  • :tabnew
    新規タブ
  • gt / gT
    次/前のタブ
Tip: :set number と :set relativenumber は移動効率が劇的に上がります。

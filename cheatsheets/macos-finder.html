macOS Finder Cheat Sheet
エンジニア向けに“作業効率が上がる Finder の操作”を整理。 ファイル操作・検索・表示切替を素早く。
macOS Finder Keyboard
基本はキーボード：ショートカットで Finder の体感速度が変わります。
エンジニア作業では「検索」と「パスの表示」が重要。
移動・表示

ウィンドウ操作

  • Cmd + N
    新しい Finder ウィンドウ
  • Cmd + W
    ウィンドウを閉じる
  • Cmd + T
    新しいタブ
  • Cmd + ↑
    上位フォルダへ

表示切替

  • Cmd + 1
    アイコン表示
  • Cmd + 2
    リスト表示
  • Cmd + 3
    カラム表示
  • Cmd + 4
    ギャラリー表示

サイドバー

  • Cmd + Option + S
    サイドバーの表示/非表示
  • Cmd + Option + P
    パスバーの表示/非表示
  • Cmd + /
    ステータスバーの表示/非表示
検索・ファイル操作

検索

  • Cmd + F
    Finder 検索を開始
  • Cmd + Option + F
    検索条件にフォーカス
  • Cmd + Shift + G
    パスで移動（Go to Folder）

ファイル操作

  • Cmd + Delete
    ゴミ箱へ移動
  • Cmd + Shift + Delete
    ゴミ箱を空にする
  • Cmd + D
    選択項目を複製
  • Enter
    名前を変更

プレビュー

  • Space
    Quick Look（プレビュー）
  • Cmd + Y
    フルスクリーンプレビュー
  • Cmd + I
    情報を見る
エンジニア向けショートカット

開発で使う

  • Cmd + Shift + .
    隠しファイル表示の切替
  • Cmd + Option + V
    移動（切り取り→貼り付け）
  • Cmd + Option + L
    ダウンロードフォルダへ

便利な設定

  • 表示設定 > タブ
    フォルダをタブで開く
  • 環境設定 > 詳細
    すべてのファイル名拡張子を表示
Tip: パスバーとステータスバーを常時表示にすると、作業中の階層確認が速くなります。
